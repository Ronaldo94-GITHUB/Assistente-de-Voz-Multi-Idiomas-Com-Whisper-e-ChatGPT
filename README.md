# 🎤 Conversando Por Voz Com o ChatGPT Utilizando Whisper e Python

## 📌 Sobre o Projeto
Este projeto integra **Whisper (Speech-to-Text)**, **ChatGPT (OpenAI API)** e **gTTS (Text-to-Speech)** em uma aplicação Python.  
O sistema permite que o usuário faça perguntas por voz, receba respostas inteligentes do ChatGPT e ouça essas respostas sintetizadas em áudio.

## 🚀 Tecnologias Utilizadas
- [Whisper (OpenAI)](https://github.com/openai/whisper) – Transcrição e tradução de áudio  
- [OpenAI API](https://platform.openai.com/docs/) – Processamento de linguagem natural  
- [gTTS (Google Text-to-Speech)](https://pypi.org/project/gTTS/) – Conversão de texto em voz  
- Python 3.9+  
- Google Colab ou Jupyter Notebook
